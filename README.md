### 线形统计图规格

##### Y轴 连续值
###### 规格
* 不能伸缩
* 不能平移
* 连续值范围根据离散点对应的Y轴值来确定

###### 相关属性
* entityMin 对应的X轴离散点的最小值
* entityMax 对应的X轴离散点的最大值
* yMin 根据entityMin/entityMax定义出来的Y轴范围的最小值
* yMax 根据entityMni/entityMax定义出来的Y轴范围的最大值
* displayMin 当前显示的Y轴范围的最小值
* displayMax 当前显示的Y轴范围的最大值

##### X轴 离散点
###### 规格
* 可伸缩
最小：显示所有的离散点
最大：显示１个离散点
* 可平移
显示的离散点数量不变
